发表于 2021-11-21

Dropout和学习率

Dropout

  1. 若模型参数太多，而训练样本太少的时候，容易出现过拟合的现象。
  2. 过拟合的表现：模型在训练数据上损失函数较小，预测准确率较高，但是在测试数据上损失函数比较大，预测准确率低
  3. 解决方法之一：模型集成，训练多个模型进行组合，但是费时
  4. Dropout有效缓解过拟合，一定程度上达到正则化的效果，在训练过程中，随机的忽略部分神经元，在正向传播过程中，这些被忽略的神经元对下游神经元的贡献效果暂时消失。在反向传播时，这些神经元也不会有任何权值的更新。
  5. 在每个训练批次中，通过忽略一半的特征检测器（让一半的隐层节点值为0），可以明显的减少过拟合现象。减少特征检测器间的相互作用（指某些检测器依赖其他检测器才能发挥作用）
  6. 等于0.5的时候效果最好，随机生成的网络结构最多

学习率

  1. 学习率决定了参数移动到最优值的速度，学习率过大，会越过最优值，学习率过小，优化的效率可能过低，长时间算法无法收敛
  2. 学习率应该随着训练的进行逐渐衰减，刚开始使用较大的学习率，使结果快速收敛，训练进行中逐步降低学习率和收敛的速度
  3. 线性衰减、指数衰减
  4. Learning Rate = Learning Rate * 1/(1 + decay * epoch)
  5. Learning Rate = Initial Learning Rate * Drop Rate^(floor[(1 + Epoch)/Epoch Drop])，decay为衰减率，epoch为迭代数。
